About Us

Welcome to Parks Are Awesome

Our mission is to promote the amazing parks located throughout the United States and raise awareness about the problems that they may face. There is so much beauty to be found in the wildlife and plantlife that live in parks. Our hope is that people of all ages will use our service to connect with nature and enjoy parks. Parks Are Awesome provides users with an interface to find local parks and search for nearby community events and volunteering opportunities taking place. We also provide users with a database of the flora and fauna native to regions throughout the U.S.

What are you waiting for? Find an awesome park near you today!

Restful API: https://documenter.getpostman.com/view/4706698/RWMCt9fN

Tools used:

Postman - API tool to get data from our data sources and document our API.

Amazon Web Services - Used for hosting our website.

HTML/CSS - Used to develop the front end of the website.

Flask - Framework used to host our backend and front-end templates.

jQuery - used for querying calls to our database and API.

SQLAlchemy - used for creating our database and retrieving data from it.

Docker - used to containerize during development and deploy our front-end and backend servers.

React - used to render our front-end components in real time.

Data sources:

Meetup.com - Found events related to outdoors/saving the environment and the api gave us all the information for that particular event, which we used for our Volunteer ops section.

NPS.gov - Information gathered by finding the park code for a particular park we wanted information on and using an api call to get the name and description of the park.

Wikipedia.org - Found a particular article about wildlife and used the wikipedia api to obtain a description and image of a particular plant or animal.

Yelp.com - Found rating information for each park using Yelp Fusion API.
